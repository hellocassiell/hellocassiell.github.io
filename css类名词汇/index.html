css类名词汇

Posted on 2017-11-18 • In css

css 命名技巧及规范， 是构建健壮页面的基础。

1 词汇量

css 易于属性使用，难于取名。
总结在场景， 结构， 框架中的词汇量
center 居中 加上它就可以
header body footer 经常用于结构的表达，缩写
hd bd ft
item

2 命名的规则

最忌讳按它的外表， 功能， 或更差的是按它的意义来取名。
组件（常用的界面模块，70%日常工作都有组件）
结构 html结构分层 少于4层， css选择器后代选择器

空格 利用结构， 体现重要的css 布局信息。
.weui-cells>.weui-cell 单复数 .items>.item
div UI块 -> Block -> Element
.weui-cell>.weui-cellhd+.weui-cell__bd+weui-cell__ft

3 BEM

Block 块 前面会加命名空间
cells cell
Element 子元素
__ 两个下划线 降级
Modifier 修饰
